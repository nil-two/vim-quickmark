INTRODUCTION					*quickmark-introduction*

*vim-quickmark* is a Vim plugin to provide an operator
to open marked path at any time.

Latest version:
https://github.com/nil2nekoni/vim-quickmark


==============================================================================
INTERFACE					*quickmark-interface*

------------------------------------------------------------------------------
KEY MAPPINGS					*quickmark-key-mappings*

<Plug>(quickmark-mark)				*<Plug>(quickmark-mark)*
			Mark current path.

<Plug>(quickmark-open)				*<Plug>(quickmark-open)*
			Open marked path.

<Plug>(quickmark-tab)				*<Plug>(quickmark-tab)*
			Open marked path with new tab.
